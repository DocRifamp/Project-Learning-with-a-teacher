**Проект: Обучение с учителем**

**Описание:** «В один клик» осуществляет розничную торговлю посредством интернет-платформы, реализуя различные категории товаров.По результатам  анализа операционной деятельности за отчётный период выявлена негативная динамика ключевых показателей, характеризующих покупательскую активность целевой аудитории. Необходимо удержать постоянных клиентов, предлагая им товары по их вкусу.

**Задача:**  разработать решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность.

Для реалиазции проекта предоставлено 4 дата сета. `market_file.csv` , `market_money.csv`, `market_time.csv`, `money.csv`.

План работы следующий

1.  Загрузка данных
2.  Предобработка данных
3.  Исследовательский анализ данных
4.  Объединение таблиц
5.  Корреляционный анализ
6.  Создание пайплайна для поиска оптимальной модели
7. Анализ важности признаков
8. Сегментация покупателей
9. Общие выводы
